Arduino Weather Information

This is the web interface for my weather station. The data for the web interface is collected from an Arduino and is transferred serially to a NodeMCU ESP8266 which then post the information to my Rust Server. The Arduino collects data from a temperature sensor, a humidity sensor, a barometric pressure sensor, a light sensor, and an RTC module.
